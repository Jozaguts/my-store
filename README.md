## About My-store

My store is an online store focused on selling t-shirts with integration to social networks for user registration, sharing products and payment gateways such as paypal and stripe 



## Core

- Backend Laravel
- Storage Amazon S3
- FrontEnd Stack
  - VueJs 
  - Vue-Router 
  - Vuex 
  - Vuetify
